# SOC Dashboards

## Overview

Four dashboards were developed to provide comprehensive visibility into the monitored environment.

---

# Dashboard 1

SOC Overview

Displays:

- Total Alerts
- Critical Alerts
- Severity Distribution
- MITRE ATT&CK Techniques

---

# Dashboard 2

Authentication Monitoring

Displays:

- Successful Logins
- Failed Logins
- Account Lockouts
- RDP Activity

---

# Dashboard 3

Windows Security Monitoring

Displays:

- Process Creation
- PowerShell Activity
- DNS Queries
- Network Connections

---

# Dashboard 4

Threat Detection & Incident Response

Displays:

- Custom Rule Hits
- Attack Timeline
- High Severity Alerts
- Recent Security Incidents

---

# Purpose

The dashboards enable analysts to monitor security events, investigate incidents, and validate custom detections.
