# Threat Detection & Incident Response Dashboard

## Overview

The Threat Detection Dashboard consolidates security alerts generated by custom Wazuh detection rules.

It provides incident visibility, attack timelines, MITRE ATT&CK mapping, and supports security investigations.

---

## Visualizations

- Critical Alerts
- High Severity Alerts
- Custom Rule Hits
- Attack Timeline
- Top Custom Rules
- MITRE ATT&CK Coverage
- Targeted Endpoints
- Recent Security Incidents

---

## Security Value

- Monitor high-risk security events
- Validate custom detection rules
- Investigate attack activity
- Track incident trends
- Support incident response workflows
