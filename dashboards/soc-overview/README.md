# SOC Overview Dashboard

## Overview

The SOC Overview Dashboard provides a high-level summary of the security posture across the monitored environment. It enables analysts to quickly identify abnormal activity, prioritize alerts, and monitor overall system health.

---

## Purpose

This dashboard serves as the primary landing page for Security Operations Center (SOC) analysts by presenting key security metrics and trends in a single view.

---

## Visualizations

- Total Alerts
- Critical Alerts
- High Severity Alerts
- Active Agents
- Alerts Timeline
- Alerts by Severity
- Top Alert Rules
- Top MITRE ATT&CK Techniques

---

## Security Value

- Measures overall alert volume
- Highlights critical security events
- Identifies the most frequently triggered detection rules
- Displays attack trends over time
- Provides visibility into MITRE ATT&CK coverage
