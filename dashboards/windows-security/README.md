# Windows Security Monitoring Dashboard

## Overview

This dashboard provides visibility into endpoint activity collected through Sysmon and Windows Event Logs.

It enables analysts to monitor process execution, PowerShell usage, network activity, and other endpoint telemetry.

---

## Visualizations

- Process Creation
- PowerShell Activity
- Network Connections
- DNS Queries
- Process Timeline
- Top Executed Processes
- Recent Sysmon Events

---

## Security Value

- Detect suspicious processes
- Monitor PowerShell abuse
- Analyze endpoint activity
- Investigate network behavior
- Support threat hunting
