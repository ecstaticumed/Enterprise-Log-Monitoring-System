# Authentication Monitoring Dashboard

## Overview

The Authentication Monitoring Dashboard focuses on Windows authentication events and account activity.

It helps detect brute-force attacks, account lockouts, unauthorized logins, and suspicious authentication behavior.

---

## Visualizations

- Successful Logins
- Failed Logins
- Account Lockouts
- RDP Logins
- Authentication Timeline
- Top Targeted Users
- Top Source IPs
- Recent Authentication Events

---

## Security Value

- Detect brute-force attacks
- Identify compromised accounts
- Monitor RDP access
- Investigate authentication failures
- Track user login activity
